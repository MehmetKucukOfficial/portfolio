# Mehmet Küçük — Portfolio

Personal portfolio website built with Next.js 16 and Tailwind CSS v4.

**Live:** [portfolio-orpin-kappa-33.vercel.app](https://portfolio-orpin-kappa-33.vercel.app)

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Deployment:** Vercel

## Sections

- **Hero** — Introduction with typewriter effect
- **Projects** — 6 featured projects with visuals and GitHub links
- **Skills** — Frontend, Backend, AI/ML, Mobile, Database, Tools
- **Contact** — Contact form and social links

## Projects Featured

| Project | Category | Tech |
|---|---|---|
| Voice Project — Konuşmacı Tanıma | AI / ML | Python, scikit-learn, SVM, MFCC |
| Hospital Management System | Web | Node.js, Express, MySQL, Socket.io |
| NewsApp — Android Haber Uygulaması | Mobile | Kotlin, MVVM, Retrofit, Room DB |
| PCAT — Fotoğraf Galeri | Web | Node.js, MongoDB, Mongoose |
| JobFinder — İstihdam Platformu | Web | Node.js, MongoDB, bcrypt |

## Run Locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)
